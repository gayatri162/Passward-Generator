# 🔐Password Generator

A simple and responsive password generator built with HTML, CSS, and JavaScript.

## ➡️Features
- Generate strong random passwords
- Copy password to clipboard
- Responsive design

## 💡Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Download the project
2. Open index.html in a browser
